# Blank Extension Svelte Vite Tailwind

This is the same browser extension rebuilt as a development project with Svelte, Vite, and Tailwind.

## Install

```powershell
npm install
```

## Build

```powershell
npm run build
```

Load this folder in Chrome after build:

```text
C:\Users\hadadx\Desktop\blank-browser-extension-svelte-vite-tailwind\dist
```

## Work Files

- `src/App.svelte` - main extension UI and logic
- `src/app.css` - Tailwind entry and base popup sizing
- `public/manifest.json` - Chrome extension manifest
- `public/icons` - copied icons from the old extension
- `public/pages` - copied page fragments from the old extension
