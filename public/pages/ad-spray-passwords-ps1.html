Using Spray-Passwords.ps1

Spray-Passwords.ps1 queries Active Directory through LDAP to identify active domain users, then performs LDAP authentication attempts to determine whether a supplied password is valid for each account.

Advantage

It may generate less network noise than SMB-based password spraying, but the failed authentication attempts can still be logged, detected, and trigger account lockouts.

Check Password Against All Admin Users

.\Spray-Passwords.ps1 -Pass Nexus123! -Admin
