impacket-wmiexec

wmiexec is an Impacket tool that leverages Windows Management Instrumentation (WMI) to execute commands remotely using valid credentials or NTLM hashes. It creates processes through the Win32_Process class and returns command output over SMB without installing a service, making it a common tool for lateral movement.

Requirements

  • Valid credentials or an NTLM hash.
  • Membership in the Administrators group on the target host.
  • The Windows Management Instrumentation (Winmgmt) service running.
  • Network access to TCP 135 + dynamic RPC ports (WMI/DCOM) and TCP 445 (SMB).
  • Firewall rules allowing WMI/DCOM and SMB traffic.

Remote Session
