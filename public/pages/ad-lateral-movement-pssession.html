PSSession

A PowerShell Session (PSSession) is a persistent remote PowerShell connection established over Windows Remote Management (WinRM). It allows authenticated users to execute commands, run scripts, and manage remote systems within an interactive or reusable session. PSSessions are commonly created using New-PSSession and accessed with Enter-PSSession or Invoke-Command.

Create Remote Session

Define User Credentials

$username = 'jen';
$password = 'Nexus123!';

Convert Password To SecureString

$secureString = ConvertTo-SecureString $password -AsPlaintext -Force;

Create PSCredential Object

$credential = New-Object System.Management.Automation.PSCredential $username, $secureString;

Create PSSession

New-PSSession -ComputerName <target ip> -Credential $credential

Use The Session

Enter-PSSession 1
