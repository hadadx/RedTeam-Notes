mimikatz + psexec

Overpass-the-Hash upgrades a stolen NTLM hash into a Kerberos Ticket Granting Ticket (TGT). Mimikatz loads the hash into a new logon session, a domain action triggers issuance of a TGT, and PsExec then reuses the resulting Kerberos tickets to gain remote code execution as the target user without performing NTLM authentication over the network.
